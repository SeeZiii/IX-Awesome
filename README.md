# IX-Awesome
IX A bukan sekadar ruang dengan meja dan papan tulis. Di sini waktu terasa cepat, seolah belum siap berpisah. Kami pernah ramai, sepi, kacau, tapi tetap bersama. Tawa yang dulu biasa kini mahal diulang. Obrolan kecil jadi kenangan. IX A jadi rumah singkat kami, dan kini hanya tinggal jejak saat kami harus melangkah pergi. Selamanya di hati kita....
